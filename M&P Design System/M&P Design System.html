Do
  • Always show a typing indicator before an agent or bot reply lands.
  • Truncate conversation list previews to a single line with ellipsis.
  • Show timestamps in 12-hour format (10:32 AM) — localise for market.
  • Use the product card message type to surface relevant items in-thread.
  • Provide a clear 'Retry' affordance on failed messages with an error badge.
Don't
  • Don't show read receipts in broadcast or order-notification threads.
  • Don't disable the input without showing a clear reason (e.g. 'Chat closed').
  • Don't render full-size images inline without a loading skeleton state.
  • Don't use red for the agent bubble — it implies error or urgency.
  • Don't allow messages to span more than 72% of the container width.
