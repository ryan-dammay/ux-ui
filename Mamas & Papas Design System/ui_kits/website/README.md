# Mamas & Papas — Website UI Kit

A high-fidelity recreation of the Mamas & Papas e-commerce website UI.

## Notes
- Reconstructed from brand research (no codebase access)
- Uses Poppins (Google Fonts) — confirmed brand font per guidelines
- Uses Lucide icons as substitute for M&P's custom icon set
- Colors and layout based on public brand guidelines research

## Screens in index.html
1. Homepage (hero + featured categories + product grid)
2. Product detail page (PDP)
3. Navigation mega-menu
4. Basket drawer
