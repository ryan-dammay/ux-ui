# Bloomingdale's Website UI Kit

A click-through prototype of the Bloomingdale's e-commerce website.

## Screens
- **Home** — hero banner, category strip, product grid, promo banner
- **PLP** (Product Listing Page) — sidebar filters, product grid, sort
- **PDP** (Product Detail Page) — image gallery, color/size selector, add to bag
- **Bag** — line items, order summary, promo code
- **Wishlist** — saved items with add-to-bag actions
- **Account** — order history, nav sidebar

## Usage
Open `index.html` in a browser. Click products/nav to navigate between screens.

## Design Rules Applied
- Font: Inter Variable (from `../../fonts/`)
- Colors: from `colors_and_type.css` token system
- Buttons: 2px radius, black primary, grey hover
- No emoji, no pill buttons, no gradients on UI chrome
- Cards: 1px #ECECEC border, 2px radius
- Magenta (#FA16C8) used only for hover states and active wishlist
